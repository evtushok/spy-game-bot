# Шпион (SpyFall) Бот

Telegram-бот для популярной настольной игры "Шпион" (SpyFall). Игроки получают карточки с каким-либо персонажем (предметом, локацией), но один из них — шпион, который должен вычислить где они находятся, не раскрыв себя.

SpyFall Bot — это интерактивная многопользовательская игра, где участники через вопросы и ответы пытаются вычислить шпиона. Идеально для вечеринок, командных встреч и онлайн-дружбы.

## Правила игры

### Как играть:

1. **Старт игры** — Какой-то участник компании на своём устройстве создает игру
2. **Раздача ролей** — Участник показывает каждому игроку его карточку
3. **Один игрок** получает карточку "ШПИОН"
4. **Фаза вопросов** — Игроки по кругу задают друг другу вопросы (или же говорят факты о выпавшем слове)
5. **Цель обычных игроков** — Вычислить шпиона, не раскрывая загаданное слово
6. **Цель шпиона** — Понять, что это за предмет (персонаж, локация и так далее) и не быть пойманным
7. **Голосование** — После обсуждения игроки голосуют, кто шпион
8. **Разоблачение** — Если шпион угадал слово, он побеждает

### Победа:

- **Обычные игроки** побеждают, если правильно угадали шпиона
- **Шпион** побеждает, если:
  - Его не разоблачили на голосовании
  - Он правильно угадал локацию

## Использование

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/evtushok/spy-game-bot.git
cd spy-game-bot
```

### 2. Создайте виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Режим разработки (polling)
```bash
python bot.py
```

## Команды бота

- `/start` — Приветственное сообщение и правила
- `/help` — Подробная помощь по игре
- `/game` — Начать игру
